### ⚡ TypeSurvival

**A real-time adaptive typing game that fights back.**

Built with `Next.js 14` · `React 18` · `Upstash Redis` · `Web Workers`

- 🧠 Client-side **Trie** for sub-millisecond prefix autocomplete — zero server round-trips
- 🎯 **Levenshtein edit-distance** in Heat Mode to surface near-miss distractors
- 🚀 **Web Worker** offloads vocab search, keeping UI non-blocking at 60 fps
- 📊 Custom `useTypingMetrics` hook tracks WPM, Accuracy & KSPC in real time
- 🏆 **Upstash Redis** sorted sets for O(log n) leaderboard ranking

### 🗺️ Community Event Hub

**A full-stack community platform with geo-services.**

Built with `Python` · `Flask` · `SQLAlchemy` · `Leaflet.js` · `SQLite`

- 🔐 Role-based access control (Attendee vs Organiser) with **bcrypt** & session management
- 📍 Dual-provider geocoding pipeline (**Nominatim → Photon** fallback) for venue resolution
- 📏 **Haversine-based** proximity filter to discover events within a configurable radius
- 🗓️ Many-to-Many RSVP system with real-time attendee counts on organiser dashboard
- 🗺️ **Leaflet.js** interactive map with event pins, pop-up details, and hydrated JSON payloads
